# Super Nintendo Entertainment System

Place Super Nintendo `.sfc` and `.smc` images in this directory. Subdirectories are supported.

PixelDeck's early SNES core launches standard LoROM and HiROM cartridges without enhancement chips. 65C816 and SPC700 execution, DMA, controllers, H/V timer IRQs, save states, APU communication/timers, eight-voice S-DSP stereo audio, and an initial Mode 0/1 video renderer are active. Compatibility and timing accuracy are still limited.

Cartridges with nonstandard map headers or enhancement chips remain disabled with an explanation in the dashboard. Do not commit or redistribute commercial game images unless you have the rights to do so.
